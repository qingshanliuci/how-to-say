# How to Say — 职场表达教练

基于 Patrick Winston "How to Speak" 理念，面向职场人的 AI 表达教练。
不给模板背诵，给可复用的表达标准：什么必须做、什么必须避免、什么原则适用于什么场景。

## 两个版本

### How to Say（完整版）— 持续成长

面向**迫切想提升说话能力、愿意持续练习**的用户。

**首次使用**自动 onboarding：
- 识别你的角色（实习生/1-3年/中层）
- 了解你的老板风格（结论先行型/数据驱动型/关系优先型）
- 根据画像定制教练建议

**核心能力**：
- 8 大类 80 条原则（含英文职场场景）
- 6 大中文场景 + 4 大英文场景速查
- 个性化教练：根据你的角色和老板风格调整建议
- 老板画像建模：深度适配老板偏好
- 使用记录累积：每次练习自动记录，下次回顾
- 成长报告：每 5 次生成表达模式诊断 + 改进建议

**安装**：复制 `SKILL.md` + `principles-reference.md`

### How to Say Lite（精简版）— 快速润色

面向**只需要快速改好一段话**的用户。没有设定、没有画像、没有历史记录。

**贴原文 → 诊断 → 改写。30 秒搞定。**

**核心能力**：
- 10 条铁律逐条诊断
- 8 场景结构模板（含中英文）
- 12 句救急句型
- 8 条红线自动消除

**安装**：只需复制 `SKILL-lite.md`（单文件，无依赖）

## 版本对比

| 能力 | 完整版 | 精简版 |
|------|--------|--------|
| 表达诊断 & 改写 | 70+ 原则深度诊断 | 10 条铁律快速诊断 |
| 场景模板 | 10 场景（中+英） | 8 场景速查 |
| 稳定器句型 | 12 句 | 12 句 |
| 负面约束 | 8 条 | 8 条 |
| Onboarding 画像 | 有 | 无 |
| 老板风格适配 | 有 | 无 |
| 使用记录 | 有（累积） | 无（用完即走） |
| 成长报告 | 有 | 无 |
| 英文原则库 | 10 条详细原则 | 2 个场景模板 |
| 文件数 | 2 个 | 1 个 |

**选哪个？**
- 想持续提升 → 完整版
- 只想快速改段话 → 精简版
- 不确定 → 先用精简版，觉得好再升级

## 使用

所有版本都支持自然语言触发，不需要记命令：

**完整版触发词**：帮我准备发言、怎么说、汇报准备、面试准备、开会发言、明天开会我该怎么说、老板问起来我怎么回、这个汇报怎么写、quarterly review、1-on-1 怎么准备、更新角色、换了老板、成长报告...

**精简版触发词**：帮我润色、帮我改改、这句话怎么说更好、帮我组织一下、帮我改得专业一点...

## 原则体系

**8 大类 80 条**（完整版）：

1. 根原则：先搞清楚你为什么要开口（4 条）
2. 结构原则：没有结构就没有职业感（6 条）
3. 语言原则：短、准、稳比高级更重要（7 条）
4. 说服原则：别人不是因为你说得多才信你（6 条）
5. 职场场景原则：6 大场景 22 条（22 条）
6. 负面约束：必须硬压的坏习惯（13 条）
7. 状态原则：紧张时怎样不崩（5 条）
8. 成长原则：怎样长期把表达练出来（7 条）
9. 英文职场场景原则：1-on-1、stand-up、review、跨文化（10 条）
