# 说话总被打断？汇报完领导没反应？

你不是不会想，你是不会说。

**How to Say** 是一个装在 AI 助手里的「职场表达教练」。装上之后，你跟 AI 说话时只要提到"帮我准备发言"、"这段话怎么改"，它就自动变成你的表达教练——帮你诊断问题、改写表达、给你可以直接用的结构。

---

## 它能帮你什么

### 明天要向领导汇报，不知道怎么组织

你说：

> 帮我准备明天的周会汇报，要向领导汇报上周进度

AI 会告诉你：
- 你的汇报应该按这个结构：**进度 → 问题 → 判断 → 需要的支持**
- 第一句话就要给结论，不要铺垫
- 如果有问题要报，必须带着你的建议一起说

然后让你写一版，它逐句帮你改。

### 写了一段话，觉得不太行但说不出哪不行

你说：

> 帮我改改这段话——"作为刚入职不久的新人，在实际工作中我也发现了自身不少需要改进的地方，数据分析能力有待提高……"

AI 会告诉你：

> **问题**：三句话全在说"我不行"，没有一句带行动方案。听起来像在道歉，不像在汇报。
>
> **改成**：数据分析目前停留在罗列阶段，缺少趋势洞察。我计划 4 月完成 Excel 数据透视表和基础 SQL 自学，5 月起独立产出月度分析报告。

### 面试时被问"你最大的缺点是什么"

你说：

> 明天面试，怎么回答"你最大的缺点"

AI 会给你一个结构：

> **承认一个真实但可控的短板 → 说明你怎么看这个问题 → 给出你正在做的改进 → 展示已有的进展**

## 两个版本，选适合你的

### 🔥 精简版——"帮我快速改好这段话"

**适合**：不想折腾设定，就是有段话要改、有个场景要准备。

装上就能用，贴原文就给你改。不记录任何东西，用完就走。

### 🚀 完整版——"我想系统提升说话能力"

**适合**：觉得自己表达一直是短板，想认真练一练。

第一次用的时候会问你三个问题：
- 你是什么角色（实习生、工作几年了、还是带团队）
- 你最常遇到什么场景（汇报多还是开会多）
- 你老板是什么风格（喜欢听结论、喜欢看数据、还是在意团队感受）

之后每次用，它的建议都是根据你的情况定制的。用得越多，它越了解你——会记住你上次用了什么结构，还会每隔一段时间给你一份「表达成长报告」，告诉你进步了什么、接下来该练什么。

**不确定选哪个？先用精简版。觉得好了再升级。**

## 怎么装

### 龙虾（OpenClaw）用户

把下面这句话复制粘贴发给你的龙虾，它会自动帮你装好：

```
帮我安装这个技能 https://github.com/qingshanliuci/how-to-say
```

等龙虾回复"安装完成"，就可以直接说"帮我准备明天的汇报"了。

## 装好之后，像跟朋友说话就行

不用记任何口令。你怎么说话，它就怎么帮你。

**要准备汇报？**
> "帮我准备明天的汇报，要跟领导说上周的进度"

**有段话想改好？**
> "帮我改改这段话：……"（直接把你写的贴进去）

**面试要用？**
> "面试官问我为什么跳槽，我怎么回答比较好"

**明天要在会上发言？**
> "明天周会我想提一个问题，怎么说比较专业"

**英文场景也行：**
> "help me prepare for tomorrow's 1-on-1 with my manager"

**用了完整版的话，还可以：**
> "换了新老板" → 它会重新了解你老板的风格，调整建议
>
> "我的表达报告" → 看看你这段时间的进步和接下来该练什么

## 它背后是什么

80 条经过整理的职场表达原则，来自 MIT 教授 Patrick Winston 的 [How to Speak](https://www.youtube.com/watch?v=Unzc731iCUY) 理念，由 [井九](https://github.com/qingshanliuci) 延伸到中文职场场景。

不是让你背话术，而是教你一套判断标准：**什么时候该先说结论、什么时候该带数据、什么时候该停下来重组、什么时候该闭嘴。**

<details>
<summary>80 条原则覆盖了什么（点击展开）</summary>

| 类别 | 核心问题 |
|------|---------|
| 根原则 | 你为什么要开口？听众关心什么？ |
| 结构原则 | 先说结论还是先铺垫？怎么分层？ |
| 语言原则 | 怎么让话更短更准更稳？ |
| 说服原则 | 怎么让人信你而不是觉得你在扯？ |
| 场景原则 | 自我介绍/面试/开会/汇报/提问/回答怎么说？ |
| 负面约束 | 哪些说话习惯必须改掉？ |
| 状态原则 | 紧张时怎么不崩？ |
| 成长原则 | 怎么长期把表达练出来？ |
| 英文场景 | 1-on-1、stand-up、performance review 怎么说？ |

</details>
